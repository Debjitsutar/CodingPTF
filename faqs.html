Frequently Asked Questions

Find answers to common questions about CodeCraft Academy, our coding platform, and learning resources.

Getting Started

How do I create a developer account?

To create your developer account:

  1. Click "Sign Up" in the top-right corner
  2. Choose between email registration or GitHub login
  3. Complete your profile (select your skill level and interests)
  4. Verify your email address
  5. Start your first coding challenge!
What programming languages do you support?

Our platform currently supports:

  • JavaScript (Node.js, React, Angular, Vue)
  • Python (Django, Flask)
  • Java
  • C++
  • C#
  • Ruby
  • Go
  • Swift
  • PHP

We're constantly adding new languages based on community demand.

Is there a mobile app available?

Yes! Our mobile app is available for both iOS and Android:

  • iOS: Download from the App Store
  • Android: Available on Google Play

The mobile app supports code editing, challenges, and learning on the go with our mobile-optimized IDE.

Coding Environment

How does the online IDE work?

Our browser-based IDE features:

  • Real-time code execution with instant feedback
  • Syntax highlighting and intelligent code completion
  • Terminal access for command-line operations
  • Collaborative editing for pair programming
  • Version control integration (Git)

Example of running a simple Python program:

# Welcome to CodeCraft
print("Hello, World!")
# Click 'Run' to see the output
Can I use my own development environment?

Yes! While our online IDE is fully featured, you can:

  1. Connect your local VS Code with our Live Share extension
  2. Use our CLI tool to sync challenges with your local machine
  3. Import/export projects via GitHub

We recommend starting with our online IDE to take advantage of instant feedback and automated testing.

How do I debug my code?

Our debugging tools include:

  • Step-through debugging with breakpoints
  • Variable inspection
  • Console output
  • Error highlighting with suggested fixes

To start debugging:

// Set breakpoints by clicking line numbers
function calculateTotal(items) {
    let total = 0;
    for (let item of items) {
        total += item.price;
    }
    return total;
}
// Use the debug panel to step through execution
